# Wild Ideas: Experimental Vector Art

Unhinged creative possibilities for pen + knife + paper.

---

## Cut + Draw Hybrids

### Negative Space Drawing
Draw intricate pattern, then cut away the "background" - the paper itself becomes the negative space. Mount over colored paper. The drawing defines what gets removed.

### Window Frames
Cut rectangular/organic windows in top sheet. Draw elaborate frames/borders around each window. Layer over different drawn sheets - swap what's visible through each window.

### Woven Paper
1. Draw Pattern A on Sheet 1, cut into vertical strips
2. Draw Pattern B on Sheet 2, cut into horizontal strips
3. Weave together - Pattern A and B interleave
4. Different patterns create moiré at intersections

### Draw-Cut-Draw Sandwich
1. Draw on paper (black lines)
2. Cut intricate pattern (removes some drawn areas)
3. Place over new paper
4. Draw again through the cut holes (red pen)
5. Separate layers - you have two related pieces

### Venetian Blinds
Cut horizontal slats (don't remove - just cut top and bottom of each strip). Fold alternating strips forward/back. Draw different images on front-facing vs back-facing surfaces. Viewing angle changes what you see.

## Layered Transparencies

### Depth Mapping
Same geographic/data region drawn on multiple sheets of vellum:
- Layer 1: Coastlines and water (blue)
- Layer 2: Elevation contours (brown)
- Layer 3: Roads and infrastructure (red)
- Layer 4: Labels and annotations (black)

Stack with spacers - creates physical depth. Shadows from lighting add dimension.

### Animation Cels
Draw frames of animation on transparencies. Stack multiples for motion blur. Fan through for flipbook. Backlight for projection.

### Interference Patterns
Two transparencies with identical line patterns. Slide one over the other - moiré patterns animate. Mount in frame with sliding mechanism.

### Color Mixing
Primary colors on separate transparencies (CMY). Overlap creates full color image. Shift layers to "misregister" for glitch effects.

## Paper Engineering

### Pop-Up Mechanisms
Score fold lines (light cut or emboss), cut tabs and slots. Flat drawing transforms into 3D sculpture when opened.

Mechanisms:
- V-folds (180° → 90° pops up element)
- Parallel folds (moving platforms)
- Pull tabs (sliding reveals)
- Rotating wheels (volvelles)

### Tunnel Books (Peep Shows)
Multiple cut frames with increasing apertures. Spacers between layers. View through front - creates forced perspective tunnel. Each layer drawn with different scene depth.

### Flexagons
Cut and score hexagonal/square flexagons. Draw different patterns on each hidden face. Flex to reveal. A hexahexaflexagon has 6 hidden faces - 6 different drawings on one piece of paper.

### Kirigami Architecture
Cut + fold patterns that pop into buildings, bridges, geometric forms. Draw details before cutting. Flatten to mail, pop up to display.

### Möbius Strips
Cut long strip, draw continuous pattern, twist and join. Pattern that only exists on a one-sided surface. Draw something that "transforms" as you follow it around.

## Data Visualization Art

### Personal Data Portraits
- GPS tracks of a year's movement → one continuous line
- Music listening history → frequency bars as cityscape
- Text message timestamps → radial plot
- Sleep data → wave patterns

### Temporal Maps
Same location, multiple time periods overlaid:
- Historical coastlines (erosion/accretion)
- Urban growth rings (like tree rings)
- Seasonal variations (river levels, vegetation)

### Data Physicalization
Convert spreadsheet to physical properties:
- Row = vertical position
- Column A = horizontal position
- Column B = symbol type (cut vs draw)
- Column C = rotation
- Column D = scale

Any dataset becomes generative art.

### Network Graphs as Constellations
Social networks, code dependencies, citation graphs → draw as star maps. Node importance = star size. Edges = constellation lines. Cut out, backlight.

## Hidden / Reveal Mechanics

### Anaglyphic Secrets
Draw two overlapping images:
- Left image in red
- Right image in cyan
View through red filter → see only cyan image
View through cyan filter → see only red image
Without filter → chaotic overlay

### Scratch-Off Art
1. Draw detailed image (ink that resists scratching)
2. Cover entire surface with scratchable layer (metallic pen?)
3. Viewer scratches to reveal

Alternative: Cut scratch zones, cover with removable paper. Selective reveal.

### Blacklight Reactive
Draw with UV-reactive pen over regular drawing. Normal light: one image. Blacklight: hidden elements appear.

### Water Reveal
Draw with wax or resist, then watercolor wash. Lines appear only when wet. Dries invisible again. (May need special pens/paper)

### Heat Reveal
Thermochromic inks change at body temperature. Touch to reveal hidden elements. Winter vs summer appearance.

## Generative Systems

### Rule-Based Drawing Machines
Define rules, let them execute:
- "Line turns 90° when it hits another line"
- "Circles placed at every intersection"
- "Color changes every 100mm of travel"

Rules create emergent complexity.

### L-System Ecosystems
Multiple L-system plants growing in same space. Competition rules: if growth intersects another plant, terminate that branch. Resulting "forest" shows interaction.

### Cellular Automata Fields
Run Game of Life or other CA. Each generation = one drawing layer or color. History of evolution visible in final piece.

### Strange Attractors
Lorenz, Rössler, Chen attractors as continuous pen paths. Single line, never crosses itself, fills space chaotically. Natural single-stroke drawings.

### Fractal Boundaries
Cut along fractal curves (Koch, dragon curve). Two pieces that fit together perfectly but have infinite edge length. Draw different patterns on each piece.

## Impossible Objects

### Penrose Constructions
Draw/cut impossible triangle, endless staircase, impossible cube. Physical paper versions that work from one viewing angle.

### Anamorphic Cylinders
Wildly distorted drawing that looks correct only when reflected in cylindrical mirror placed at center. Cut circle hole for mirror placement.

### Hollow Face Illusion
Cut and fold convex face. Viewed from distance, appears to follow viewer. Draw features that enhance illusion.

### Ambigrams
Words that read the same (or different word) upside down. Generate rotational ambigram typography. Cut as stencil - works both orientations.

## Destruction / Transformation

### Designed Decay
- Perforated tear lines that invite destruction
- Patterns that "complete" only when torn
- Instructions: "Remove one section per day for 30 days"

### Burn Reveal
Draw on flash paper or specially treated paper. Controlled burn reveals what's beneath. Fire as artistic tool. (SAFETY FIRST)

### Dissolve Art
Water-soluble paper. Draw detailed image. Submerge. Document dissolution. Residue is the final piece.

### Shred and Reconstruct
Draw continuous pattern. Feed through shredder. Reassemble strips offset. Original pattern transformed.

### Compost Canvas
Draw on seed paper. Plant it. Art becomes garden. Document growth.

## Interactive / Participatory

### Exquisite Corpse Generator
System draws head section. Cover, pass to human for body. Cover, machine draws legs. Reveal complete creature. Hybrid human-machine collaboration.

### Choose Your Own Adventure Maps
Branching paths with cut flaps. Lift flap to see consequence of choice. Navigate through drawn landscape with decision points.

### Puzzle Art
Cut into interlocking jigsaw pieces. Each piece is complete mini-artwork. Assembled creates meta-artwork. Can be reassembled in different configurations.

### Modular Tiles
Square tiles that connect any edge to any edge. Draw with matching edge patterns. Infinite reconfiguration. Viewer arranges.

### Coloring Book Generation
Draw outlines and structure. Leave fills empty. Human completes with color. Machine precision, human expression.

## Sound + Music

### Waveform Landscapes
Audio waveforms become mountain ranges. Song = geography. Draw on long scroll for timeline.

### Spectrogram Cities
Frequency spectrum over time becomes city skyline. Bass = wide buildings, treble = narrow spires. Time = horizontal.

### Music Box Strips
Cut holes in paper strip that plays in music box mechanism. Generative composition. Visual pattern = audible pattern.

### Graphic Scores
Abstract notation for experimental music. Performer interprets drawing as instructions. Each drawing = unique performance.

## Serialized / Editioned Work

### Parametric Editions
Same algorithm, different seeds. Edition of 50 where each is unique but clearly related. Serial number encoded in pattern.

### Progressive States
Like printmaking states:
1. Draw base layer (edition of 100)
2. Add second layer (edition of 50 from #1)
3. Add cutting (edition of 25 from #2)
4. Add color (edition of 10 from #3)
5. Add gold leaf by hand (edition of 1 from #4)

### Degrading Masters
Each print slightly modifies the source file. First edition crisp, 100th edition degraded/evolved. Planned obsolescence as feature.

### Time Capsule Series
One piece per day/week/month for a year. Same format, responding to current events/data/mood. Calendar of a year in drawings.

## Material Experiments

### Multi-Paper Collage
Cut different paper types into same pattern:
- Tracing paper (transparent)
- Black paper (silhouette)
- Textured paper (tactile)
- Metallic paper (reflective)
- Map paper (found imagery)

Assemble into single composition.

### Embossed Lines
Score without cutting (or light pressure cut). Creates raised/recessed lines. No ink - pure paper texture. Visible only in raking light.

### Edge Painting
Cut thick stack of paper. Paint/draw on the edge. Appears when pages fanned. Different images at different fan angles.

### Paper Curls
Parallel cuts through paper. Paper between cuts curls differently based on grain direction. Draw before cutting - drawing becomes 3D texture.

## Conceptual / Absurdist

### 1:1 Scale Maps
Map of room drawn at actual size. Lay over floor. Walk on your location.

### Self-Describing Drawings
Drawing that describes its own creation process. "This line is 47mm long" drawn as 47mm line. Meta-documentation.

### Invisible Drawings
White pen on white paper. Drawing exists but is invisible. Certificate of authenticity describes what was drawn.

### Drawings of Drawings
Draw the plotter drawing a drawing of the plotter drawing a drawing... Recursive mise en abyme.

### Legal Documents
Terms of service, EULAs, contracts rendered as decorative patterns. Beautiful bureaucracy. Sign here to accept.

### Redaction Art
Draw document, then cut out "classified" sections. Remaining text creates concrete poetry from negative space.

## Scale Experiments

### Microscale
Smallest possible drawing with pen capability. Requires magnification to view. Hidden details in larger work.

### Mural Scale
Tile small pages into massive compositions. Each page standalone, together creates room-sized work.

### Scroll Format
Continuous horizontal drawing. Roll-to-roll viewing. Journey format - beginning to end. Or infinite loop.

### Accordion Books
Single continuous paper folded. Opens to full length (mural) or reads page by page (book). Different experiences same object.

## Encoding / Steganography

### QR Art
QR codes that are also aesthetic patterns. Scannable and beautiful. Hide URLs in artwork.

### Barcode Patterns
UPC codes as design elements. Each artwork has embedded price, ID, message.

### Punch Card Programs
Historical computer punch cards. Cut holes = executable program (for historical machines). Or just aesthetic reference to computing history.

### Braille Integration
Embossed Braille text as texture element. Readable by touch. Hidden message for those who read it.

### Cipher Wheels
Volvelle-style code wheels. Rotate to encrypt/decrypt messages. Functional cryptographic tool as art.

## Truly Unhinged

### Ouija Plotter
Attach pen to planchette mechanism. "Spirits" move the plotter. Generative automatic drawing. (Random number generator as spirit realm)

### Weather Drawings
Real-time weather data controls pattern. Barometric pressure = line weight. Wind = angle. Temperature = spacing. Drawing = frozen moment of weather.

### Bio-Feedback Art
Heart rate, galvanic skin response, EEG → pattern parameters. Drawing is portrait of mental/physical state during creation.

### Decay Timelapse
Draw same image repeatedly. Ink runs out, pen wears, paper changes. Document degradation over hundreds of iterations.

### Competitive Drawing
Two plotters, one paper. Different goals. Must negotiate space. Collision = damage. Survival of the fittest algorithm.

### Sleep Drawings
Body movement during sleep → coordinates. Draw path of restless night. Each night = unique fingerprint.

### Destruction Instructions
Beautifully rendered detailed instructions on how to destroy the piece. Following instructions destroys instructions.
